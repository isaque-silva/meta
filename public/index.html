API & Integração

Endpoints para automação de deduções

POST /api/deducoes

Deduz um valor da meta aberta mais recente do funcionário identificado pelo usuário. Útil para disparar automaticamente a cada erro operacional detectado por outro sistema.

Request body

{
  "usuario": "joao.silva",
  "percentual": 10,
  "mes_ano": "08/2026",
  "motivo": "Erro de lançamento no pedido #1234"
}

Exemplo cURL

curl -X POST http://localhost:3000/api/deducoes \
  -H "Content-Type: application/json" \
  -d '{"usuario":"joao.silva","percentual":10,"mes_ano":"08/2026","motivo":"Erro"}'
